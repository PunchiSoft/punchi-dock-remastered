# Fase 3: consolidación de skills

Se diferenciaron y mejoraron las skills solapadas del proyecto.

- Se definieron como canónicas `qml-ui-creation`, `js-plasma-backend`, `kde-plasma-6-dev`, `ki18n-localization`, `plasmoid-packaging` y `kde-sdk-reference`.
- Los seis nombres genéricos quedaron como alias temporales para preservar compatibilidad.
- `ui-design`, `ux-review`, `visual-review` y `accessibility` recibieron procedimientos y límites distintos.
- Se corrigieron supuestos que no correspondían a la estructura real del repositorio.

La evidencia y el mapa de transición están en `docs/plandeaccion/fase-3-consolidacion-skills.md`.
