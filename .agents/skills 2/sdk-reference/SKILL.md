# Sdk Reference Skill

## Instrucciones Detalladas
Esta skill se centra exclusivamente en: Consulta y uso de la documentación oficial del SDK de Plasma y Qt.
1. Mantén el enfoque únicamente en resolver problemas relacionados con sdk-reference.
2. Consulta la documentación oficial relevante (KDE/Qt/Plasma) cuando sea necesario.
3. No combines responsabilidades: si necesitas cambiar algo fuera del ámbito de sdk-reference, delega o activa la skill correspondiente.

## Checklist
- [ ] ¿Los cambios se limitan estrictamente al ámbito de sdk-reference?

## Buenas Prácticas
- Priorizar la modularidad y el bajo acoplamiento.

## Criterios de Aceptación
